# Personal Portfolio Website Development Prompt

## Project Overview
Create a modern, interactive personal portfolio website using Next.js 14+, Tailwind CSS, and Lenis for smooth scrolling animations. The portfolio should be visually stunning, performant, and showcase professional skills effectively.

- [ ] Scroll indicator with smooth animation

## Technical Requirements

### Tech Stack
- **Framework:** Next.js 14+ with App Router
- **Styling:** Tailwind CSS with custom components
- **Animations:** Lenis + GSAP/Framer Motion
- **Type Safety:** TypeScript
- **Icons:** Lucide React or Heroicons
- **Fonts:** Custom Google Fonts with optimization
- **Deployment:** Vercel/Netlify with CI/CD

### Development Guidelines
- [ ] Component-based architecture with reusable components
